# Two-Node Radar Hardware Photos

Store dated photos for both Node A and Node B. Each photo set should show:

- Node label and ESP32 STA MAC
- BU01 antenna orientation
- ST7789 wiring and display orientation
- GY-511 axis orientation relative to display top
- MAX30102 placement
- I2C, default SPI, and dedicated HSPI wiring
- Battery, charger, regulator, switch, and decoupling

Do not label new photos as Master or Slave. Historical photos may keep their original labels if the caption marks them as pre-pivot evidence.
